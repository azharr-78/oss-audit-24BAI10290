##  Project Overview
Welcome to my OSS Capstone Project! For this audit, I chose to dive into **Python**—a language that fundamentally powers the modern AI and Machine Learning stack, and one that is shaped entirely by its community. 

This repository contains the five practical shell scripts required for the technical portion of the audit. They demonstrate core Linux command-line skills, system administration, and bash scripting concepts like loops, conditionals, and file I/O.

##  Prerequisites & Environment
To run these scripts, you will need:
* A Linux-based operating system (tested on Ubuntu 24.04 via WSL).
* Standard bash shell.
* The `dpkg` package manager (used in Script 2 and for Script 4's log file).
